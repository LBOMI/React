#### React-Slick
 + 슬라이드 기능을 구현하기 위해 *React-Slick*이라는 라이브러리를 사용했다. https://react-slick.neostack.com/
 + 이 라이브러리를 활용하여 <ins>자동 슬라이드</ins>, <ins>화살표 커스텀</ins>, <ins>페이지 인덱스</ins> 등을 구현할 수 있다.
